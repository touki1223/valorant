valorantとは

valorantは、5人対5人で行うFPSゲームです。
スパイクと呼ばれる爆弾を設置・起爆する攻撃側（アタッカー）とそれを阻止する防御側に分かれ、
基本最大25ラウンド（12ラウンドで攻守交代、13ラウンド先取で勝利）を攻防する。

【勝利条件】

攻め 守り
・守り側の全滅
・スパイクを設置した後でのスパイク爆発 		・攻め側がスパイクを設置する前に全滅させる
・ラウンドのタイムアップ
・攻め側が設置したスパイクの解除

【ラウンドの流れ】

  1. 購入・準備フェーズ

    2. ラウンドが始まると、まず武器やアビリティーを購入する。その後は進行可能範囲で準備をする時間。攻め側は開幕に攻めるラウンドを決め、守り側は敵が攻めてきても良いようにアビリティーを設置し守りを固めよう。

  2. 戦闘フェーズ

    3. 購入フェーズが終わると、100秒間の戦闘フェーズが始まる。ここではそれぞれの陣営の勝利条件を目指し、戦っていく。

  3. 決着が付くまで1~2を繰り返す

    4. 13ラウンド先取で決着が付くまで1~2を繰り返す。攻守陣営は12ラウンド経過で交代し、12:12になるとコンペティティブならオーバータイムに、アンレートならサドンデスに突入する。
